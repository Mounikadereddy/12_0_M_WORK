--- The scripts folder:

-- RBPS Client
The rbpsClient.sh file and the RbpsClient.jar in the scripts folder 
need to be copied to be copied by the administrator to a bin 
directory in the RBPS user's home directory.  

The rbpsClient.sh script needs to be set to run in cron/control-M every 1 hour.
The endpoint of the RBPS web service is a property inside the shell script.

-- Letter Generation
The rbps-functions.sh and rbps-vva.sh files in the scripts folder need 
to be copied by the administrator to a bin directory in the RBPS user's 
home directory.  The RBPS users's home directory needs to be writable 
by those scripts.

The rbps-vva.sh script needs to be set to run in cron/control-M once a day.

The rbps-vva.sh script will be given the name of the environment in which it 
is to run as the only parameter.  It uses that name in the path to the 
'/AdobeDoc/${ENV}/RBPS' directory where the pdf and csv files are to be found.   
This should be the same path as the 'rbps.lettergen.homedir' 
entry in the rbps.properties file.
